Introduction to the Linux command line interface

Command Execution Environment

▼▲
Information available at the time of running a command.
  • Open files and redirections in effect.
  • Current working directory.
  • Inherited functions and variables.
  • Shell options.
  • Builtins and functions have access to aliases.
▲

The Environment

▼▲
Information passed on to the executed command.
  • A list of name=value pairs explicitly exported to the executed command.
  • The shell's initial environment give or take any modifications, additions or removals made to it.
